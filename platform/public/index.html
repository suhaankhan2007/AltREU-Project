Field guide

How to read a light curve

  • Time (x) Each point is one night. Left = earlier, right = later.
  • Brightness (y) Up = brighter. A microlensing event is a temporary brightening — a hump.
  • Baseline The star's steady brightness when nothing is happening.

Astronomers measure brightness in magnitudes (smaller = brighter). Here we always orient so up = brighter.

The three shapes

Microlensing ✦

  • One smooth, symmetric hump — rises and falls once.
  • Returns to baseline and stays. The signal we're hunting.

Variable star

  • Repeats — a periodic rhythm, no isolated event.
  • A pulsating or eclipsing star — not lensing.

Noise / junk

  • Scattered points, no coherent shape.
  • Glitches or a faint target — mark as noise.
